# Recognizer

## Setup and run

create a `python3.10` virtual environment by typing the following command in the terminal:

```bash
python3.10 -m venv .venv
```

Then install all dependencies by:

```bash
pip install -r requirements.txt
```

Afterwards run:

```bash
python main.py
```

```bash
uvicorn app:app --port 8000
```
